# FitnessTracker

This Angular project is a web application designed to provide a tracking of your fitness goals. It has been developed during the course [Angular (Full App) with Angular Material, Angularfire & NgRx](https://www.udemy.com/course/angular-full-app-with-angular-material-angularfire-ngrx/), from [Maximilian Schwarzm√ºller](https://www.udemy.com/user/maximilian-schwarzmuller/).

It has been made using `Angular 16.2.0`, and provide a fake database using a json file `db.json`.

### Development server

Run `npm start` to start the application at `http://localhost:4200/`.
Run `npm run serve:exercises` to start the fake backend, provided at  `http://localhost:3000`.
